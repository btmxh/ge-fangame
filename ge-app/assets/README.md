# Asset credits

## Fonts

- [ProggyClean](https://github.com/bluescan/proggyfonts) (currently unused).
- [Pixeloid](https://ggbot.itch.io/pixeloid-font) (planned as main font).

## BGM

- `ambient-bgm` is a rearrangement of Shirakami Fubuki's
  [Glow Embrace](https://www.youtube.com/watch?v=BWEmA6AxIyM&list=RDBWEmA6AxIyM)
  (soty 2025 winner yay).
- `menu-bgm` is a very crude rearrangement of Tsunomaki Watame's
  [Kitto](https://www.youtube.com/watch?v=dVAf4gRZro0&list=RDdVAf4gRZro0)
  (only a loop from the instrumental).

## Textures

Currently, all are hand-drawn using [Aseprite](https://www.aseprite.org/).
